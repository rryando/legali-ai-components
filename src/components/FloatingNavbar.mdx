# Floating Navbar

A responsive navbar that transforms from a full-width header to a floating pill on scroll.

## Usage

```tsx
import { FloatingNavbar } from 'legali-ai-components';

function App() {
  return (
    <>
      <FloatingNavbar />
      <main>
        {/* Content */}
      </main>
    </>
  );
}
```

## Props

<Controls />

## Behavior

- **Top**: Full width, transparent background.
- **Scroll**: Transforms to a floating pill with glassmorphism effect (blur + semi-transparent background).
- **Mobile**: Adapts to a simplified view with a hamburger menu.
